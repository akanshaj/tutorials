Data Structures

Data structure defines how the data will be stored in memory and what
operations will be applicable on it. 

Data structure allows us to write better programs by helping us manage data in
more effecient manner.

Types of data structures

1. Trivial (Basic)
2. Linear
	- Data exists in continous memory locations
	- Eg : Array, Stack, Queue
3. Non-linear
	- Data may or may not exist in continous memory locations
	- Eg : Linked List, Tree, Graph

Operations
==========

1. Insertion
2. Deletion
3. Traversing
4. Searching
5. Sorting
